Тестовое задание для Angular-разработчика:
- Реализовать генератор форм из произвольного json (массив объектов)
- Собрать форму из компонентов тестового макета.
- Для каждого поля формы передаются название комненента, label, placeholder, имя модели для дальнейшей передачи на бэк,
данные необходимы для рендера (например для select список из чего выбираем)
- Компоненты могут и должны повторяться (например input может использоваться для имени и фамилии)

Комментарии:
- Макет сохранился только в pdf (поэтому вёрстка делается просто похожая, сильно на ней не зацикливаемся)
- Естественно, чтобы не отнимать много времени у разработчика, можно ограничиться 2-3 элементами формы (например: select, input, checkbox)
- Важен сам подход, а не объём проделанной работы
